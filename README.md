# 🏃‍♂ Nike Landing Page

A modern and responsive *Nike Landing Page* built using *HTML* and *CSS*.  
This project showcases a clean and minimal design inspired by Nike’s branding, focusing on product display and elegant typography.

---

## 🚀 Features

- ✅ Fully Responsive (works on desktop, tablet, and mobile)
- 🎨 Clean, modern UI using *Inter* Google Font
- 👟 Product showcase with hover effects
- 🛒 Navigation bar with search, cart, and menu icons
- 💡 Simple scrollable product section
- ⚡ Fast, lightweight, and pure HTML + CSS (no frameworks)

---

## 🧱 Technologies Used

- *HTML5*
- *CSS3*
- *Remix Icons* (for icons)
- *Google Fonts – Inter*

---

## 📁 Project Structure
```bash
Nike-Landing-Page/
│
├── index.html # Main HTML file
├── style.css # CSS styles and responsive media queries
├── images/ # Folder containing Nike product images
└── README.md # Project documentation
```

---

## 📱 Responsive Design

The layout automatically adapts to all screen sizes:
- *Desktop (≥1024px)* – Full layout with navigation menu
- *Tablet (≤1024px)* – Adjusted spacing and font sizes
- *Mobile (≤768px)* – Stacked layout, hidden menu list, horizontal product scroll

---

## 🧩 How to Use

1. Clone or download this repository.
2. Make sure the images folder and style.css are in the same directory as index.html.
3. Open index.html in your browser.
